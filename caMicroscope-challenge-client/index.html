Select an image by clicking at Choose Image below and preview will be shown. After that click on Upload button. This will send file to server where the RGB channels will be seprated and once server is done with processing the request, it will respond with URL for three images for Red, Green and Channel. Finally the images are shown on canvas and combined again in browser. Note the images might be appeared blurred due to fact that we are using canvas to show images.
